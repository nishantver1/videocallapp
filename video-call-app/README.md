# Video Call App

A real-time video calling web application built with **React, Node.js, Socket.io, and WebRTC**.

## Live Demo

**Frontend:** https://videocallapp-blond.vercel.app

##  Features

* Real-time video calling
* Dynamic meeting rooms
* Peer-to-peer connection using WebRTC
* Screen sharing
* Microphone toggle
* Camera toggle
* End call functionality
* Automatic room exit when a participant leaves
* Modern responsive UI with Tailwind CSS

##  Tech Stack

### Frontend

* React
* Vite
* Tailwind CSS
* Socket.io Client
* Lucide React

### Backend

* Node.js
* Express
* Socket.io

### Real-Time Communication

* WebRTC
* Google STUN Server

##  Installation

### Clone the repository

```bash
git clone https://github.com/nishantver1/videocallapp.git
```

### Install dependencies

#### Client

```bash
cd client
npm install
npm run dev
```

#### Server

```bash
cd server
npm install
node index.js
```
